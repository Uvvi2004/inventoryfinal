# Inventory App (Flutter + Firestore)

## Features
- Add items
- Real-time updates using StreamBuilder
- Update items
- Delete items
- Total item count (Enhanced Feature 1)
- Clear all items button (Enhanced Feature 2)

## Tech Used
- Flutter
- Firebase Firestore

## How It Works
The app uses Firestore as a cloud database. Data is streamed in real-time using StreamBuilder, so any changes reflect instantly in the UI.
